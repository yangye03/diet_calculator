# 饮食追踪 PWA

一个简洁实用的健身饮食记录应用，帮助你追踪每日碳水化合物和蛋白质摄入量。

## 功能特点

- 📊 实时显示今日剩余可摄入量
- ⚡ 快速记录每餐营养数据
- 🎯 自由设定每日目标
- 📱 PWA支持，可添加到手机主屏幕
- 💾 本地存储，完全离线可用
- 📈 历史记录查看

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- PWA (Progressive Web App)

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 生产构建
```bash
npm run build
```

## 在iPhone上使用

1. 在Safari中打开部署后的URL
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 完成！可以像原生应用一样使用

## 部署

### Vercel（推荐）

1. 导入GitHub仓库到Vercel
2. 自动检测并部署
3. 每次推送自动更新
